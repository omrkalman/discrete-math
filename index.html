Logical Symbols Set Theory Symbols Functions and Relations Number Theory Symbols Combinatorics Symbols Graph Theory Symbols Miscellaneous Symbols
  • ∧ (and) - Conjunction
  • ∨ (or) - Disjunction
  • ¬ (not) - Negation
  • ⇒ (implies) - Implication
  • ⇔ (if and only if) - Biconditional
  • ⊕ (exclusive or) - XOR
  • ∈ - Element of
  • ∉ - Not an element of
  • ⊆ - Subset
  • ⊂ - Proper subset
  • ⊄ - Not a subset
  • ⊇ - Superset
  • ⊃ - Proper superset
  • ∅ - Empty set
  • ∪ - Union
  • ∩ - Intersection
  • ** or − - Set difference
  • × - Cartesian product
  • |A| - Cardinality of set A
  • f : A → B - Function from A to B
  • ◦ - Function composition
  • ↦ - Maps to
  • ∼ - Equivalent relation
  • ≠ - Not equal to
  • ℕ - Set of natural numbers
  • ℤ - Set of integers
  • ℚ - Set of rational numbers
  • ℝ - Set of real numbers
  • ℂ - Set of complex numbers
  • |a| - Absolute value of a
  • gcd(a, b) - Greatest common divisor
  • lcm(a, b) - Least common multiple
  • n! - Factorial of n
  • (n choose k) = C(n, k) - Binomial coefficient
  • P(n, k) - Permutations of n elements taken k at a time
  • G = (V, E) - A graph with vertices V and edges E
  • deg(v) - Degree of a vertex v
  • Kₙ - Complete graph on n vertices
  • Cₙ - Cycle graph on n vertices
  • Pₙ - Path graph on n vertices
  • ∼ or ≈ - Isomorphic graphs
  • ∀ - For all (universal quantifier)
  • ∃ - There exists (existential quantifier)
  • ∃! - There exists exactly one
  • ℤₙ - Integers modulo n
  • ≡ - Congruence (modular equivalence)
  • → or → - Tends to
